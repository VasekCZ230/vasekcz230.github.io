GraphicsGale

GraphicsGale is a bitmap and pixel art editor for creating sprites and animations. It supports frame-by-frame animation, onion skinning, layers, and palette control. The program can export animated GIFs and sprite sheets, making it popular among retro game developers and pixel artists. Developed by Humanbalance Ltd, it runs on Windows and is free to use.

HJ-Split

HJ-Split is a popular file splitting and joining utility software primarily used in Windows environments. It's designed to split large files into smaller segments and recombine them afterward.

SunVox/a>

SunVox is a modular synthesizer and pattern-based music tracker for creating electronic music. It features a flexible module system, powerful synthesis, and built-in effects. The program supports MIDI and WAV formats and can export to various audio files. SunVox runs on Windows, macOS, Linux, Android, iOS, Raspberry Pi, Windows CE, and Palm OS. It is free for Windows, macOS, Linux, Windows CE, Palm OS and Raspberry Pi, while the Android and iOS versions are paid.
WinDirStat

WinDirStat (Windows Directory Statistics) is a disk usage statistics viewer and cleanup tool for windows. It provides users with a visual representation of disk space usage on their computer's hard drives, allowing them to identify large files and directories that may be consuming significant storage space.
