Auto Clicker v1.2

Auto Clicker is a simple auto clicker. You can set the click delay, max click time, choose left, right, or middle mouse button, and pick a start/stop hotkey. It runs in the background and listens for your hotkey to toggle clicking. You need to have Java 8 installed! Download button is at the bottom of my home page.

Prism Launcher Offline Accout Manager v2.1.1

Prism Launcher Offline Account Manager is a tool that allows you to manage offline accounts without having to log in to your Minecraft account stored in the accounts.json file used by Prism Launcher. It allows you to open or create the file, view existing accounts, add new offline accounts using a nickname (with auto-generated UUIDs), add an empty Microsoft account (MSA), and delete selected accounts. The application automatically saves changes and ensures that the file uses the correct version of the format (3). You need to have Java 8 installed, you can find the Java download button at the bottom of the home page. !!! FOR EDUCATIONAL PURPOSES ONLY!!!

Program Launcher v2.2.0

Program Launcher is a tool ideal for portable use on USB drives or anywhere you need quick access to your apps. It makes launching programs easy from one place. You can add programs, organize them into categories, and set custom icons in png, gif (animated), ico, or exe formats. All program and icon paths are saved in a config file, there are also instructions how to use it.
