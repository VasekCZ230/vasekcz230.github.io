Applet Launcher v2.0

Applet Launcher is a tool for easily running Java applets from .class or .jar files. You can load an applet, set custom parameters, control window size, and choose network modes. All output is shown in a built-in console. You can also automate launching via the command line by specifying the applet path, class name, parameters and other settings. You need to have Java 8 installed, you can find the Java download button at the bottom of the home page.

Prism Launcher Offline Accout Manager v2.1.1

Prism Launcher Offline Account Manager is a tool that allows you to manage offline accounts without having to log in to your Minecraft account stored in the accounts.json file used by Prism Launcher. It allows you to open or create the file, view existing accounts, add new offline accounts using a nickname (with auto-generated UUIDs), add an empty Microsoft account (MSA), and delete selected accounts. The application automatically saves changes and ensures that the file uses the correct version of the format (3). You need to have Java 8 installed, you can find the Java download button at the bottom of the home page. !!! FOR EDUCATIONAL PURPOSES ONLY!!!

Program Launcher v2.1.2

Program Launcher is a tool ideal for portable use on USB drives or anywhere you need quick access to your apps. It makes launching programs easy from one place. All program and icon paths are saved in a config file, there are also instructions how to use it.
